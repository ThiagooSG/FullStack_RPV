1. Calcule a porcentagem entre 2 números.  
*Informações*  
Exemplo: 25% de 40 é 10;  
Fórmula de porcentagem: (y / x) * 100;  
Uso da função:  
let x = 40;  
let y = 10;  
let pct = calcPct(x, y);  
console.log(`${pct}% de ${x} é ${y}`);
